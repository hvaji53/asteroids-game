# Asteroids Game

A browser-based Asteroids-style game built with HTML Canvas and JavaScript.

## Features

- Player-controlled spaceship
- Asteroid spawning
- Projectile shooting
- Collision detection
- Lives system
- Score counter
- Game over screen with final score
- Projectile and Asteroid cleanup
